About Flow-Graph

Flow-Graph is a data visualization web-app intended for analyzing flow cytometry data exported from FlowJo. Its data preprocessing tools target FlowJo-specific formatting challenges, but it is flexible enough to accommodate any CSV-formatted dataset containing categorical and continuous variables.

Key Features

  • CSV File Upload: Simple drag-and-drop or file selection interface
  • Data Processing: Split sample identifiers into separate columns and automatically clean FlowJo export formatting
  • Interactive Visualization: Create boxplots and bar charts with customizable grouping
  • Faceted Graphs: Compare across multiple experimental conditions
  • Session-Based: Data is processed in isolated sessions for privacy

Technology Stack

Flow-Graph is built with Python Flask backend, using pandas for data manipulation, plotnine (Python implementation of ggplot2) for visualization, and SQLite for temporary data storage. The frontend uses vanilla JavaScript with some Bootstrap styling.

Source Code

Privacy & Data Retention

Your uploaded data is stored temporarily. Data is automatically deleted 2 hours after storage or whenever you upload new data during the same session. You can also manually delete your data at any time using the "Delete Data" option in the navigation bar.
